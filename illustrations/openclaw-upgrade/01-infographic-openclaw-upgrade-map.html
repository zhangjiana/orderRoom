一张图看懂 OpenClaw 现在该升到哪一版

把版本、变化和升级判断放在一起看，比单纯盯着“最新号是多少”更有用。

v2026.3.24

能力扩展版。重点是 OpenAI 兼容网关补齐 /v1/models、/v1/embeddings，再加上 Skills UI、容器命令和 Node 版本预检。

v2026.3.28

稳定收口版。重点是 Qwen 迁移、provider 兼容性、按模型冷却、MCP/channels bridge、插件运行时和 memory 修复。

main 2026.3.30

主分支版本号已经继续往前走，但截至 2026年3月31日，还不是新的 tagged release。

建议升级

还停在 v2026.3.24 或更老、用 Qwen、依赖插件/MCP/memory、或者最近遇到 provider 兼容问题的用户。

可以等等

当前环境稳定、不用 Qwen、也没踩到 Skills UI、gateway、memory 这些点的用户，可以等下一个稳定 tag。

别追 main

生产环境不要直接追 main 2026.3.30。它代表开发主线在继续前进，不代表已经适合直接上线。
